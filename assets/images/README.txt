Logo Plouf — tête de vache

Pour utiliser VOTRE image de tête de vache à la place du dessin par défaut :

  1. Copiez votre image (celle que vous m’aviez fournie) dans ce dossier.
  2. Nommez-la exactement :  vache_plouf.png

L’app affichera cette image en la réduisant à la taille du logo (barre d’app, etc.).
Si le fichier n’existe pas, le logo dessiné (vache + tonneau) s’affiche à la place.
